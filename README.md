# Cocos 资源体积审计工具

用于调查 Cocos 游戏项目中资源文件的大小、分类和引用情况，帮助开发者评估包体与资源优化优先级。

## 初始目标

- 扫描 Cocos Creator 项目资源目录。
- 按资源类型分类，例如贴图、音频、动画、模型、场景、Prefab、材质、字体、视频和数据文件。
- 输出文件名、目录、大小和分类。
- 每个分类内按文件大小从大到小排序。
- 后续支持基于 `.meta` UUID 和场景/Prefab/动画等文件的引用分析。

## 推荐输出

- HTML 报告：用于人工审查。
- CSV 报告：用于表格筛选和排序。
- JSON 报告：用于自动化流程或 CI 集成。
